# Dark Mode Guide

All new pages must use semantic theme colour classes (bg-theme-bg, text-theme-primary, etc.) instead of hardcoded arbitrary values. This ensures dark mode works automatically on every new page without additional work. Never use bg-[#FAFAF8], bg-[#1A1A1A], or bg-[#F2F2EF] in new components — use bg-theme-bg, bg-theme-dark, bg-theme-alt instead. The only hardcoded colours allowed in new pages are the sage green (#8FBF8A), gold (#C9A96E), and white text on coloured backgrounds.
